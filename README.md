Portfolio Test 🌐💻

A modern personal portfolio website built to test and improve frontend web development skills through practical project development and responsive design implementation.

🚀 Features

- 🎨 Responsive Modern UI
- 📱 Mobile-Friendly Design
- ⚡ Smooth Navigation
- 🧑‍💻 Project Showcase Section
- 📄 About Me Section
- 📬 Contact Section
- 🌙 Clean Layout and Styling
- 🔥 Interactive User Experience



🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript



📂 Project Structure

| index.html | style.css | script.js | assets/



💡 Project Purpose

This project was created as a hands-on frontend development practice project to improve skills in:

- Responsive web design
- UI/UX structuring
- Layout development
- JavaScript interactivity
- Project organization

The goal was to build a clean and functional portfolio website while gaining more practical coding experience.
